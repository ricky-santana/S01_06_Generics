# S1-06
**Generics**
## 📌 Enunciado
En esta tarea profundizaremos en uno de los conceptos más potentes y flexibles del lenguaje Java: los tipos genéricos (Generics).

Los genéricos nos permiten escribir código más reutilizable, seguro y limpio, evitando conversiones de tipos manuales y reduciendo los errores en tiempo de ejecución.

Trabajarás con clases y métodos genéricos, aprenderás a limitar los tipos genéricos mediante interfaces o superclases, y pondrás en práctica el uso de varargs y de bounded types (tipo acotados). Todos los ejercicios están pensados para hacerte entender la potencia de esa funcionalidad en escenarios reales.

Qué aprenderás:
Crear métodos genéricos que acepten argumentos de cualquier tipo.
Combinar objetos genéricos con tipos específicos, incluyendo la limitación con interfaces o clases.
Utilizar varargs genéricos y descubrir sus implicaciones.
Aplicar restricciones (`bounded types`) para controlar qué tipos de objetos se pueden pasar a un método genérico.
Reforzar la relación entre genéricos, interfaces y polimorfismo.
